# 🏥 Healthcare Analytics Dashboard (Power BI)

An integrated medical diagnostic dashboard providing deep insights into **Heart Disease** trends and **Breast Cancer** diagnostics.

## 📊 Key Insights
* **Cardiovascular Analysis:** Visualized heart disease prevalence, cholesterol trends by age, and chest pain severity.
* **Oncology (Breast Cancer) Diagnostics:** Analyzed tumor morphology (Radius vs Texture) and identified malignancy patterns using advanced visualizations.

## 🛠️ Features
* **Interactive Slicers:** Synced filters for Age, Gender, and Diagnosis across pages.
* **Advanced DAX:** Custom measures for Disease Rates, Cancer Risk %, and Patient counts.
* **Modern UI:** Dark-themed professional layout with intuitive navigation.

## 📁 Files in this Repository
* `*.pbix`: The main Power BI dashboard file.
* `data.csv`: Breast Cancer dataset.
* `heart.csv`: Cardiovascular dataset.
